# Use Interesting Fonts

Typography instantly signals quality. Avoid using boring, generic fonts.

## Never Use
- Inter
- Roboto
- Open Sans
- Lato
- Default system fonts

## Recommended Fonts

### Code Aesthetic
- JetBrains Mono
- Fira Code
- Space Grotesk

### Editorial
- Playfair Display
- Crimson Pro

### Technical
- IBM Plex family
- Source Sans 3

### Distinctive
- Bricolage Grotesque
- Newsreader

## Pairing Principles

High contrast = interesting:
- Display + Monospace
- Serif + Geometric Sans
- Variable font across weights

## Sizing Guidelines

Use extremes:
- Weights: 100/200 vs 800/900 (not 400 vs 600)
- Size jumps: 3x+ (not 1.5x)

## Implementation

1. Pick one distinctive font
2. Use it decisively
3. Load from Google Fonts
